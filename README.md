# Artificial Projects - E-commerce Automation Platform

A modern, AI-powered e-commerce automation platform website built with Next.js, React, and Framer Motion. This platform helps e-commerce brands scale their operations without adding headcount through intelligent automation solutions.

### Modern UI/UX
- **Glassmorphism Navigation**: Sticky navbar with glassmorphic effect that adapts on scroll
- **Smooth Animations**: Framer Motion animations throughout the site for engaging user experience
- **Parallax Effects**: Scroll-triggered parallax animations on portfolio images
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop devices
- **Gradient Themes**: Consistent blue gradient color scheme (#0079F2 to #00d4ff)

### Key Components

#### Hero Section
- Animated gradient text with "Scale Your E-commerce Operations"
- Interactive background elements with mouse tracking
- Customer statistics card with 17k+ customers showcase
- Call-to-action buttons for getting started

#### Services & Solutions
- **Customer Support Automation**: AI-powered 24/7 support systems
- **Operations Automation**: Streamlined workflows and order processing
- **Marketing Automation**: Scalable campaigns without team expansion
- **Analytics & Insights**: Real-time dashboards and AI-driven insights

#### Portfolio/Results
- Showcase of real outcomes with animated number displays
- Parallax scroll effects on project images
- Results including: 75% cost reduction, 3x faster processing, 2x ROI increase

#### Contact Section
- **Animated Number Display**:

  - Hover/Future state: Shows potential with positive percentage and larger numbers
  - Auto-animation on mobile devices (every 1 second)
- **Social Media Integration**: Instagram, Telegram, WhatsApp, and Email links
- **Call-to-Action**: Direct phone call button
